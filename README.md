# 🕵️ Deepfake Detection of Images

An AI-powered Deepfake Detection System that analyzes facial images and predicts whether an image is **Real** or **Fake** using a trained Deep Learning model.

The project features a modern web interface built with HTML, CSS, and JavaScript, along with a Python Flask backend for image analysis.

## 🚀 How It Works

1. Upload an image.
2. Image is preprocessed.
3. Deep Learning model analyzes facial features.
4. Prediction is generated.
5. System displays:

   * Real / Fake Verdict
   * Confidence Score
   * Probability Analysis
